# CLAUDE.md — Instruções para a Equipe de Marketing com IA

Este é o projeto de marketing da marca. Leia os arquivos de contexto relevantes antes de executar qualquer tarefa.

## Estrutura de Pastas

- /contexto → contém todos os arquivos importantes de contexto da marca e do negócio. Leia os arquivos de contexto relevantes antes de executar qualquer tarefa.
- /dados → contém dados de desempenho, exportações de análises e conjuntos de dados para fins de análise.
- /exemplos → contém exemplos de referência e conteúdo de amostra usados para treinar habilidades em padrões de qualidade.
- /saidas → todos os entregáveis gerados devem ser salvos aqui, organizados por tarefa ou nome de campanha. Se não existir, crie a pasta.
- /POP → contém procedimentos operacionais padrão (POPs) e documentação de processos para quaisquer tarefas ou operações internas de marketing.

## Regras Gerais

1. Sempre leia os arquivos em /contexto antes de criar qualquer conteúdo.
2. Salve todos os entregáveis finais em /saidas.
3. Mantenha o tom e a voz da marca conforme descrito em /contexto/voz-da-marca.md.
4. Ao criar novas habilidades ou fluxos de trabalho, salve-os dentro da pasta .claude/skills/.
5. Crie novas pastas conforme necessário para organizar o trabalho.

## Como Usar Esta Estrutura

### Para criar conteúdo:
1. Leia /contexto/contexto-da-marca.md
2. Leia /contexto/voz-da-marca.md
3. Crie o conteúdo seguindo as diretrizes
4. Salve o resultado em /saidas/[nome-da-campanha]/

### Para análise de dados:
1. Coloque os dados brutos em /dados/
2. Execute a análise
3. Salve o relatório em /saidas/relatorios/

### Para novos processos:
1. Documente o processo em /POP/
2. Use como referência futura
